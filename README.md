## Introduction

Scrume is a native iOS application built with SwiftUI to manage Scrum projects.
It helps teams organize backlogs, manage sprints, and track progress efficiently.

---

## Features

* Project and sprint management
* Product backlog with user stories
* Scrum board (To Do / In Progress / Done)
* Burndown and velocity tracking
* Team management
* Works completely offline

---

## Tech Stack

* SwiftUI
* Swift 5.9
* CryptoKit (AES-256 encryption)
* Keychain (secure key storage)

---

## Architecture

* MVVM (Model - View - ViewModel)

---

## Installation

