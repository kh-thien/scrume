## Data Storage & Security

### Local Storage Only

Scrume operates in a **completely offline mode**. Your data is:

- Stored locally in the app's sandboxed container
- Never uploaded to any server
- Never shared with third parties
- Never accessible by us or anyone else

### Encryption

Your data is protected with industry-standard security:

- **AES-256-GCM Encryption**: All data is encrypted at rest using Advanced Encryption Standard with 256-bit keys in Galois/Counter Mode
- **iOS Keychain**: Encryption keys are stored securely in the iOS Keychain, protected by the device's Secure Enclave (on supported devices)
- **Complete File Protection**: Data files use iOS Complete File Protection, making them inaccessible when the device is locked
